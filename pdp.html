🃏
Spaced Repetition System (SRS) — Anki Protocol
New vocab → review after 1 day → 3 days → 7 → 14 → 30. Target 20 new cards/day from Academic Word List (570 words). Mistake cards go back to Day 1 cycle.
Retention ≥ 85%
🔬
Error-First Learning Loop
After every mock test: extract wrong answers → classify error type (grammar / vocab / time / comprehension) → drill ONLY that error type for 3 days before next full test. Never revisit what's already correct.
Targeted uplift
🪞
Shadowing Protocol (Speaking)
15 min/day: listen to BBC 6 Minute English → pause → repeat sentence at same pace, recording yourself. Compare prosody, stress, linking sounds. Weekly self-score on TED criteria: fluency, pronunciation, coherence.
S ≥ 6.0 target

How — Goal 02 · CompTIA Network+

Learning Strategies & Tracking

Score = Conceptual Depth × Practice Volume × Simulation Hours
Core Learning Formula
Network+ Readiness Formula
Exam Score = (Domain Mastery × 0.6) + (Simulation Practice × 0.25) + (Recall Speed × 0.15)
Domain MasteryAvg % correct per domain on practice exams
SimulationPBQ (performance-based questions) score in labs
Recall SpeedAnswer under 90 sec avg — tested in timed drills
🗺️
Domain-First Deep Dive (N10-009)
Study domains in weight order: Networking Concepts (23%) → Network Implementation (19%) → Network Operations (17%) → Network Security (20%) → Network Troubleshooting (21%). 1 domain per phase, never skip to next until ≥75% on domain practice test.
Pass threshold: 720/900
🧪
Hands-On Lab Protocol (GNS3 / Packet Tracer)
Each concept → build a lab scenario, not just memorize. Subnetting: calculate 50 subnets by hand before moving on. VLAN setup, routing protocols, firewall ACLs — must configure in lab, screenshot results, annotate what changed and why.
Lab hours: 20+ total
⚡
Feynman Concept Compression
After studying each sub-domain, write a 5-sentence explanation as if teaching a non-technical person. If you can't simplify it, you don't understand it. Review these summaries weekly — they become your rapid recall sheet before the exam.
Depth check
🔥
Weakness Amplification Drilling
Track domain scores in each practice exam. Any domain below 65%: add a dedicated 30min daily session on that domain only for 1 week, then retest. Never let a weak domain coast — it drags the total score down disproportionately on CompTIA's weighted scoring.
Floor: ≥ 70% per domain

How — Goal 03 · Delegation Skills

Learning Strategies & Tracking

Growth = Reflection Frequency × Intervention Courage × Feedback Loops
Core Behavioral Change Formula
Delegation Mastery Model
Leadership Score = (Clarity of Instruction × Trust Level) ÷ Micromanagement Tendency
ClarityTask assigned with: WHO does WHAT by WHEN with WHAT definition of done
Trust Level1.0 = check-in every step → 3.0 = outcome-only check
MicromanagementNumber of unsolicited follow-ups — target: 0 per assigned task
🎯
"Ask First" Rule (W1 Antidote)
Before absorbing any task that could belong to someone else, ask: "Can this be done by someone on the team? What would they need to succeed?" If yes → delegate with the SMART task format. Only absorb if no one else can do it at all.
Baseline: 1.5/5 → Target: 4/5
📋
SMART Task Delegation Format
Every delegated task must include: Specific output, Measurable criteria, Authority level (can decide X without asking me), Resources available, and Time deadline. Template: "By [date], please deliver [output], defined as [done criteria]. You have authority to [scope]. Ask me if [blocker type]."
Clarity framework
🪃
Post-Project After-Action Review (AAR)
After every project: answer 4 questions in writing — (1) What did I absorb that I should have delegated? (2) What did I delegate well? (3) What was the outcome for the delegatee? (4) What would I do differently? Score yourself 1–5 on each.
Per project
🗣️
Peer Feedback Collection Protocol
After each group project, send a 4-question anonymous survey to teammates: (1) Clarity of task assignment 1–5, (2) Did I listen before deciding 1–5, (3) Did I respect your working pace 1–5, (4) What should I do differently. Average score = delegation KPI.
Target avg ≥ 4/5
